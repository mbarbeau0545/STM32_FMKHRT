HAL_HRTIM_Init
Initialise le module HRTIM (configuration globale, activation des horloges, etc.).

HAL_HRTIM_MspInit
Fonction d'initialisation au niveau matériel qui configure les GPIO, IRQ, etc. (souvent appelée par HAL_HRTIM_Init).

HAL_HRTIM_WaveformTimeBaseConfig
Configure la base de temps (période, prescaler, mode de comptage, etc.) pour un timer donné (ex. Timer A).

HAL_HRTIM_WaveformCompareConfig
Configure la valeur de comparaison utilisée pour définir la largeur d'impulsion dans le cycle PWM.

HAL_HRTIM_WaveformOutputConfig
Configure les paramètres de la sortie PWM (polarité, mode à l'état inactif, mode rapide, etc.).

HAL_HRTIM_WaveformOutputStart
Démarre la sortie PWM sur un canal défini.

HAL_HRTIM_WaveformCounterStart
Démarre le compteur du timer (permettant de lancer la génération continue de PWM).

HAL_HRTIM_SimplePWMStart
Une fonction simplifiée pour démarrer rapidement une PWM avec une configuration par défaut, adaptée aux applications ne nécessitant pas une configuration fine.

HAL_HRTIM_SwapTimerOutput
Permet de forcer la mise à jour (swap) des registres « shadow » vers les registres actifs, afin d'appliquer les modifications de configuration au bon moment dans le cycle PWM.

HAL_HRTIM_FaultModeCtl
Permet d'activer ou de désactiver les entrées de faute (Fault Inputs), qui servent notamment à protéger le système en cas d'anomalies (surintensité, surtension, etc.).

HAL_HRTIM_EventConfig
Configure les événements (triggers) du HRTIM qui peuvent être utilisés, par exemple, pour synchroniser le déclenchement de l'ADC ou d'autres périphérique
